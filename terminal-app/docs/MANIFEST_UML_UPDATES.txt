PERFORMANCE CHARACTERISTICS DOCUMENTED
================================================================================

Terminal State Operations:
  - scroll_up(): O(1)
  - scroll_down(): O(1)
  - set_visible_lines(): O(1)

Output Buffer Operations:
  - add_line(): O(1) amortized
  - scroll operations: O(1)
  - trim_if_needed(): O(n) when buffer > 10,000 lines

Render Operations:
  - render(): O(visible_lines) for drawing

All documented in SCROLLING_ARCHITECTURE.md

================================================================================
EDGE CASES DOCUMENTED
================================================================================

1. Empty Buffer + Scroll Down: No effect (clamped to 0)
2. Buffer Smaller Than Viewport: scroll_position stays at 0
3. Terminal Resize Larger: May expose more content (scroll_position clamped)
4. Terminal Resize Smaller: scroll_position clamped to new max_scroll
5. Buffer Trim During Scroll: scroll_position adjusted proportionally
6. Auto-scroll When Scrolled Up: Respects user scroll position
7. Auto-scroll At Bottom: Automatically follows new output

All detailed in SCROLLING_ARCHITECTURE.md
